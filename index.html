My Projects

Open-PREC

Open-PREC is a personal project developed for the community of a popular Source Engine game, Team Fortress 2. The project itself is a plugin for the Source Engine, which enables players to automatically capture gameplay footage, organize their recordings by date and time, and collect statistics about the recorded gameplay to make later review easy.

While other similar projects exist, the motivation for my version was to create a plugin that would function on all major platforms. Other recording tools work well on Windows, but tend to exclude other platforms such as OS X and Linux. Open-PREC aims to be compatible with all of these platforms, excluding nobody from such a useful tool.

Gameboy Rust

Gameboy Rust is a project with two purposes: To test my understanding of the Rust programming language, and to learn more about hardware emulation. The project was started more than a year ago, and uploaded to GitHub in November of 2015.

Since then, I have learned a lot about both emulation in general, and about the specifics of the Nintendo Gameboy hardware. Although currently the emulator is slow, and with limited functionality, I hope to take what I have learned and improve the project incrementally. Once the emulator is able to successfully run some of the early Gameboy games, I feel the project will be complete, and I will be able to move on. Until then, Gameboy Rust remains open source, serving as a reference to all those looking to better understand either the Rust programming language, or the Nintendo Gameboy.
